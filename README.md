# Project Detail
Survey app sample project with laravel livewire

# Setup
- git clone https://github.com/sefaiyi/captanomy.git
- cd project_folder
- composer install
- npm install
- cp .env.example .env
- php artisan key:generate
- update database variable in .evn file
- php artisan migrate
- php artisan db:seed // for sample data
- npm run dev
- php artisan serve
